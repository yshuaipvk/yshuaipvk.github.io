2.链式法则

链式法则是关于复合函数导数的性质：

如果某个函数由复合函数表示，则该复合函数的导数可以用构成复合函数的各个函数
的导数的乘积表示。

例如$z=(x+y)^2$,由$z=t^2$和$t=x+y$两个式子构成则有：

$\frac {\partial z}{\partial x} = \frac{\partial z}{\partial t} \frac {\partial t}{\partial x}$ $\frac {\partial z}{\partial t} = 2t$ $\frac {\partial t}{\partial x} = 1$ $\frac {\partial z}{\partial x}= 2t·1=2(x+y)$

3.反向传播

3.1 加法节点反向传播

以$z=x+y$为例:

$\frac {\partial z}{\partial x}=1$ ; $\frac {\partial z}{\partial y}=1$

因此，加法节点反向传播只乘以1，输入的值原封不动的流向下一个节点

3.2 乘法节点的反向传播

以$z=xy$为例:

$\frac {\partial z}{\partial x}=y$ ; $\frac {\partial z}{\partial y}=x$

乘法的反向传播将上游的值乘以传播时出入信号的翻转值。
对于$ 10 \times 5=50$,反向传播时，如果从上游传来1.3，则$1.3 \times 5 = 6.5$，$1.3 \times 10 = 13$

5.激活函数层的实现

5.1 ReLu层

激活函数ReLu表达式为：

$y=\begin{cases} x & \ (x>0)\\ 0 & \ (x\leqslant 0) \end{cases}$

y关于x的导数为：

$\frac {\partial y}{\partial x} =\begin{cases} 1 & \ (x>0)\\ 0 & \ (x\leqslant 0) \end{cases}$

正向传播时出入x大于0，则反向传播会将上游的值原封不动的传给下游；若果正向传播时x小于等于0，则反向传播中传给下游的值为0。一般假定forward和backward的参数都是NumPy数组。

1.神经网络学习的步骤

1.1 mini-patch

从训练数据中随机选出一部分数据，这部分数据成为mini-batch。学习的目标是减小mini-batch的损失函数的值。

1.2 计算梯度

为了减小mini-batch的损失函数的值，需要求出各个权重参数的梯度。梯度表示损失函数的值减小最多的方向。

1.3 更新参数

将权重参数沿梯度方向进行微小更新。

1.4 重复

重复步骤1、步骤2、步骤3。

2.两层神经网络的类

  • params：保存神经网络的参数的字典型变量
  • grads：保存梯度的字典型变量
  • init :进行初始化
  • predict（self, x）:进行识别，参数x是图像数据
  • loss(self, x, t):计算损失函数的值，x是图像数据，t是正确的标签
  • accuracy(self, x, t): 计算权重参数的精度
  • gradient(self, x, t):计算权重参数的梯度

1.Agent和环境

  Agent通过传感器感知环境并通过执行器对所处环境产生影响。人类Agent有眼睛、耳朵和其他器官等传感器，也有手、腿、声道等作为执行器。机器人Agent则可能用摄像头、红外测距仪作为传感器，各种马达作为执行器。软件Agent接受键盘敲击、文件内容和网络数据包作为传感器输入，并以屏幕显示、写文件和发送网络数据包为执行器来用于环境。
  真空吸尘器agent：世界上只有两个地点：方格A和方格B。吸尘器agent可以感应到当前处于哪个方格，该方格是否有灰尘。它可以向左移动，向右移动，吸尘，或者什么也不做。该Agent函数列表下入所示：

2.好的行为：理性的概念

理性Agent是做事正确的Agent，从概念上交，Agent函数表格的每一项都填写正确。

2.1理性

  • 定义成功标准的性能度量。
  • Agent对环境的先验知识。
  • Agent可以完成的行动。
  • Agent截止到此时的感知序列。

对每一个可能的感知序列，根据已知的感知序列提供的证据和Agent具有的先验知识，理性Agent应该选择能使其性能度量最大化的行动。

2.2全知者、学习和自主性

  • 理性不等于完美
  • 理性不要求全知

3.环境的性质

3.1任务环境规范描述

  对于吸尘器Agent，性能度量、环境、以及Agen的执行器和传感器都属于任务环境。
PEAS:性能（Performance）、环境（Environment）、执行器（Actuators）、传感器（Sensors）
对于自动驾驶出租车Agent：

Agent Type Performance Measure Environment Actuators Sensors

3.2任务环境的性质

  完全可观察的与部分可观察的：如果Agent的传感器在每个时间点都能获取环境的完成状态，即完全可观察的。噪音、不精确的传感器，或者传感器丢失部分状态数据，可能导致环境称为部分观察可观察的。
  单Agent与多Agent：独自玩字谜游戏的Agent处于单Agent环境；下国际象棋的Agent处于双Agent环境。
  确定的和随机的：如果环境的下一个状态完全区取决于当前状态和Agent执行的动作，那么该环境是确定的；否则是随机的。
  片段式和延续式：在片段式的任务环境中，Agent的经历被分为一个个原子片段。在每个片段中Agent感知信息并完成单个行动，下一个片段不依赖与以前的片段中采取的行动。在延续式的环境中，当前决策会影响到所有未来的决策。
  静态的和动态的：环境在Agent计算的时候会变化，那么该Agent的环境是动态的；否者环境是静态的。
  离散的与连续的
  已知的与未知的

4.Agent的结构

  AI的任务是设计Agent程序，它实现的是把感知信息映射到行的Agent函数。假设该程序要在某个具备物理传感器和执行器的计算装置上运行，即成为体系结构。Agent=体系结构+程序

4.1Agent程序

  Agent程序都具有同样的框架：输入为传感器得到的当前感知信息，返回的是执行器的行动抉择。Agent程序以当前感知为输入，Agent函数是以整个感知历史作为输入。Agent程序只把当前感知作为输入是因为从环境无法得到更多的信息。

四种基本的Agent程序：

  • 简单反射Agent
  • 基于模型的反射Agent
  • 基于目标的Agent
  • 基于效用的Agent

4.2 简单反射Agent

  简单反射Agent基于当前的感知行动选择心动，不关注历史。

4.3基于模型的反射Agent

  处理部分可观测的最有效的途径是让Agent跟踪记录现在看不到的那部分世界。即，Agent应该根据感知历史维持内部状态，从而至少反映出当前状态看不到的信息。

当Agent顺时针转动方向盘的时候，汽车会右转，或者在沿着高速公路向北行驶5分 钟后汽车通常应该离5分钟红前北方大约5英里远。这种关于“世界如何运转”的知识———无论是用简单的布尔电路还是用完备的科学理论实现————都被称为世界模型。使用这种模型的AGent被称为基于模型的Agent。

4.4基于目标的Agent

  知道当前的环境状态对于决策而言并不够。例如，在路口，出租车可以左转，右转，或者直行。正确的决定要取决于出租车要去哪里，即除了当前状态的描述，Agent还需要目标信息来描述想达到的状态，例如，要达到乘客的目的地。

4.5基于效用的Agent

  仅靠目标在很多环境中不足以生成高品质的行为。例如，有很多序列可以让出租车到达目的地，但有些更快、更安全、更可靠，或者更便宜。

MNIST数据集

MNIST是机器学习领域最有名的数据集之一，被应用于从简单的实验到发表的论文研究等各种场合。实际上，在阅读图像识别或机器学习的论文时，MNIST数据集经常作为实验用的数据出现。MNIST数据集是由0 到9 的数字图像构成的。训练图像有6 万张，测试图像有1万张，这些图像可以用于学习和推理。MNIST数据集的一般使用方法是，先用训练图像进行学习，再用学习到的模型度量能在多大程度上对测试图像进行正确的分类。

1.电化学阻抗的含义

给电化学系统输入一个扰动函数X，它就会输出一个响应信号Y。用来描述扰动与响应之
间关系的函数，称为传输函数$G(\omega)$。 可用如下公式描述：

如果X为角频率为$\omega$的正弦电流信号，Y为正弦电势信号，此时$G(\omega)$称为阻抗(impedance)，用Z表
示。如果X为电势信号，Y为电流信号，$G(\omega)$称为导纳(admittance)，用Y表示.阻抗和导纳称为阻纳
(immittance),用G表示，阻抗导纳互为倒数关系。

2.简单电路

以给正弦电势信号为例：
给定的为正弦电势信号 $e = Esin\omega t$

得到的为正弦电流信号 $i = Isin(\omega t+\phi)$

在阻抗谱中，我们得到的最原始的数据为正弦电流信号，具体来说是电流信号的幅值和相位

2.2 电阻

对于纯电阻 $\phi = 0 $
根据欧姆定律:
$e = iR$ $\Longrightarrow$ $i = \frac{E}{R}sin(\omega t)$

写成复数：$Z_C = R$

实部：${Z}’_R = R $

虚部：${Z}”_R = 0 $

在Nyquist图上为横轴上一个点

2.3 电容

对于电容 $\phi = \frac {\pi}{2}$

$e=Esin\omega t$

$i=\frac {dQ}{dt}= \frac{d(CU)}{dt}=C \frac{d}{dt}(Esin \omega t)
= E \omega C cos \omega t = E \omega C sin(\omega t +\frac {\pi}{2})$

$i = \frac {E}{X_C}sin(\omega t +\frac {\pi}{2})$

$X_C= \frac {1}{\omega C}$

写成复数：$Z_C=-jX_C=-\frac {j}{\omega C}$

实部：${Z}’_C = 0 $

虚部：${Z}”_C = -\frac {1}{\omega C}$

Nyquist图上为与纵轴重合的一条直线

2.4 电感

$i= Isin \omega t$

$e_L=-L \frac {dI}{dt}=-L \frac {d}{dt}(Isin \omega t) =
-I \omega t sin(\omega t +\frac {\pi}{2})$
$U_L=-e_L=Isin(\omega t+ \frac {\pi}{2})$

$Z=j \omega L$

  • Nyquist中上为与纵轴负轴重合的一条直线

2.5电阻R和电容C串联

串联电阻的阻抗是各串联元件阻抗之和：

$Z = Z_R + Z_C = R - \frac {j}{\omega C}$

实部：${Z}’ = R$

虚部：${Z}” = \frac {1}{\omega C}$

Nyquist图上为与横轴交于R与纵轴平行的一条直线

2.5 电阻R和电容C并联的电路

并联电路对阻抗的倒数是各并联元件阻抗倒数之和

$\frac{1}{Z} = \frac {1}{Z_R}+\frac{1}{Z_C}=\frac{1}{R}+j\omega C$

$Z = \frac{R}{1+(\omega RC)^2}-j\frac{\omega R^2C}{1+(\omega RC)^2}$

实部：${Z}’ = \frac{R}{1+(\omega RC)^2}$

虚部：${Z}”=- \frac{\omega R^2 C}{1+(\omega RC)^2}$

消去$\omega$ ,整理得：$({Z}’ - \frac{R}{2})^2 + {Z}”^2 = (\frac {R}{2})^2$

在Nyquist图中为圆心为(R/2,0),半径为R/2的圆

2.6 阻抗谱测到的信号

  • Parallel capactitance and dissipation factor (Cp-D) (并联电容-损耗因子)
  • Series capacitance and resistance(Cs-Rs) (串联电容-电阻)
  • Series capacitance and dissipation(Cs-D) （串联电容-损耗因子）
  • Resistance and reactance(R-jX) (电阻-电抗)
  • Impandance and phase angle(Z-theta) (阻抗-相位角)
