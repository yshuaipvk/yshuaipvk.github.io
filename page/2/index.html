发表于 2020-03-10 更新于 2020-03-15 分类于 人工智能：一种现代方法

1.Agent和环境

  Agent通过传感器感知环境并通过执行器对所处环境产生影响。人类Agent有眼睛、耳朵和其他器官等传感器，也有手、腿、声道等作为执行器。机器人Agent则可能用摄像头、红外测距仪作为传感器，各种马达作为执行器。软件Agent接受键盘敲击、文件内容和网络数据包作为传感器输入，并以屏幕显示、写文件和发送网络数据包为执行器来用于环境。
  真空吸尘器agent：世界上只有两个地点：方格A和方格B。吸尘器agent可以感应到当前处于哪个方格，该方格是否有灰尘。它可以向左移动，向右移动，吸尘，或者什么也不做。该Agent函数列表下入所示：

2.好的行为：理性的概念

理性Agent是做事正确的Agent，从概念上交，Agent函数表格的每一项都填写正确。

2.1理性

  • 定义成功标准的性能度量。
  • Agent对环境的先验知识。
  • Agent可以完成的行动。
  • Agent截止到此时的感知序列。

对每一个可能的感知序列，根据已知的感知序列提供的证据和Agent具有的先验知识，理性Agent应该选择能使其性能度量最大化的行动。

2.2全知者、学习和自主性

  • 理性不等于完美
  • 理性不要求全知

3.环境的性质

3.1任务环境规范描述

  对于吸尘器Agent，性能度量、环境、以及Agen的执行器和传感器都属于任务环境。
PEAS:性能（Performance）、环境（Environment）、执行器（Actuators）、传感器（Sensors）

3.2任务环境的性质

  完全可观察的与部分可观察的：如果Agent的传感器在每个时间点都能获取环境的完成状态，即完全可观察的。噪音、不精确的传感器，或者传感器丢失部分状态数据，可能导致环境称为部分观察可观察的。
  单Agent与多Agent：独自玩字谜游戏的Agent处于单Agent环境；下国际象棋的Agent处于双Agent环境。
  确定的和随机的：如果环境的下一个状态完全区取决于当前状态和Agent执行的动作，那么该环境是确定的；否则是随机的。
  片段式和延续式：在片段式的任务环境中，Agent的经历被分为一个个原子片段。在每个片段中Agent感知信息并完成单个行动，下一个片段不依赖与以前的片段中采取的行动。在延续式的环境中，当前决策会影响到所有未来的决策。
  静态的和动态的：环境在Agent计算的时候会变化，那么该Agent的环境是动态的；否者环境是静态的。
  离散的与连续的
  已知的与未知的

4.Agent的结构

  AI的任务是设计Agent程序，它实现的是把感知信息映射到行的Agent函数。假设该程序要在某个具备物理传感器和执行器的计算装置上运行，即成为体系结构。Agent=体系结构+程序

4.1Agent程序

  Agent程序都具有同样的框架：输入为传感器得到的当前感知信息，返回的是执行器的行动抉择。Agent程序以当前感知为输入，Agent函数是以整个感知历史作为输入。Agent程序只把当前感知作为输入是因为从环境无法得到更多的信息。

四种基本的Agent程序：

  • 简单反射Agent
  • 基于模型的反射Agent
  • 基于目标的Agent
  • 基于效用的Agent

4.2 简单反射Agent

  简单反射Agent基于当前的感知行动选择心动，不关注历史。

4.3基于模型的反射Agent

  处理部分可观测的最有效的途径是让Agent跟踪记录现在看不到的那部分世界。即，Agent应该根据感知历史维持内部状态，从而至少反映出当前状态看不到的信息。

当Agent顺时针转动方向盘的时候，汽车会右转，或者在沿着高速公路向北行驶5分 钟后汽车通常应该离5分钟红前北方大约5英里远。这种关于“世界如何运转”的知识———无论是用简单的布尔电路还是用完备的科学理论实现————都被称为世界模型。使用这种模型的AGent被称为基于模型的Agent。

4.4基于目标的Agent

  知道当前的环境状态对于决策而言并不够。例如，在路口，出租车可以左转，右转，或者直行。正确的决定要取决于出租车要去哪里，即除了当前状态的描述，Agent还需要目标信息来描述想达到的状态，例如，要达到乘客的目的地。

4.5基于效用的Agent

  仅靠目标在很多环境中不足以生成高品质的行为。例如，有很多序列可以让出租车到达目的地，但有些更快、更安全、更可靠，或者更便宜。

从光子到电流

发表于 2020-03-05 更新于 2020-03-06 分类于 太阳能电池物理

太阳辐射

  • 辐照度($P$):单位面积上接受电磁波辐射的功率密度($W/cm^2$)
  • 光谱辐照度($I$):单位面积、单位光谱能量上接受的电磁波辐射功率($W·cm^{-1}eV^{-1}$)
    辐照度$P$，是在整个光谱范围$E$内，对光谱辐照度$I$的积分：
$I = \frac {dP}{dE}$
  • 光子角通量($\beta$):单位面积、单位光谱能量、单位时间、单位立体角通过的光子数($cm^{-2}eV^{-1}s^{-1}sr^{-1}$)
  • 立体角($\Omega$):用经度($\phi$)、维度($\theta$)描述，$d\Omega = sin\theta d\theta d\phi$

发表于 2020-03-02 更新于 2020-03-04 分类于 太阳能电池物理

伏安特性曲线

  太阳能电池的伏安特性曲线表征一定光照和环境温度为$T_s=300 K$条件下，电流$I$与电压$V$的函数关系。由于电流与太阳能电池面积成正比，所以常用电流密度($J,A/cm^2$)来取代电流。

$J= \frac {I}{A}$

  太阳能电池往往用功率密度衡量电流大小：

$P(V)=J(V)V$

  加上负载电阻$R$,根据欧姆定律:

$J(V)=\frac {V}{AR}$

光生电流和量子效率

  太阳能电池在光照下的光生电流，有入射光强$bs$和太阳能电池性能共同决定。光生电流$J{ph}$和短路电流$J_{sc}$相等，可以表示为:

$J_{ph}=J_{sc}=q\int _{0}^{\infty} QE(E)b_s(E,T_s)dE $

其中，$q=1.6 \times 10^{-19}$,是电子电量；量子效率($QE(E)$)是光子能量($E,eV$)的函数,描述能量为$E$的光子，产生电子跃迁并进入外部电路的概率；$b_s$是太阳光子通量或入射光强，描述单位时间内，单位面积上，能量在$E$到$E+dE$范围内的太阳辐射光子数，$b_s(E,T_s)$与太阳温度$T_s$有关。

肖克莱方程和光生电压

  当加上负载，太阳能电池的接触电极之间形成电压$V$。由于太阳能电池的二极管特性产生暗电流，与光生电流方向相反。如果太阳能电池温度与环境温度相同，处于热平衡状态，则暗电流满足一下方程：

$J_{dark}(V)=J_0[exp(qV/k_BTa)-1]$

式中，$J_0$是反向饱和电流，$q$为电子电荷量。$k_B$为玻尔兹曼常数，$T_a$是环境温度。
  伏安特性曲线中的电流$J$,是光生电流和暗电流的叠加。

$J(V)= J_{sc}-J_{dark}(V)$ $J(V)= J_{sc}-J_0[exp(qV/k_BT_a)-1]$

  当电路断开$J(V)=0$，太阳能电池达到开路电压$V=V{oc}$,则此时光生电压$V{ph}$满足:

$V_{ph}= V_{oc}=\frac {k_BT_a}{q}ln(\frac {J_{sc}}{J_0}+1)$

由于短路电流$J_{sc}$与入射光强$b_s$成正比，那么开路电压与入射光强对数成正比。

填充因子和转换效率

  在最佳工作点，太阳能电池到达最大功率，对应最佳工作电压和最佳工作电流和最佳负载电阻，满足

$R_m = \frac {V_m}{AJ_m}$
填充因子为：
$FF = \frac {P_m}{J_{sc}V_{oc}} = \frac {J_mV_m}{J_{sc}V_{oc}} = \frac {P(V_m)}{J_{sc}V_{oc}}$

    太阳能电池由光敏电流源(IL)、二极管、串联电阻、并联电阻构成。如下图所示:

根据等效电路，可得到太阳能电池I-V表达式为：

$I=I_{ph}-I_0[exp(\frac {q(V+IR_s)}{nKT})-1]-\frac {V+IR_s}{R_sh}$

其中：$I{ph}$为光生电流，$n$为理想因子，$q$为电荷常数，$K$为玻尔兹曼常数，$T$为温度，$R_s$为串联电阻，$R{sh}$为并联电阻，$I$为输出电流，$V$为输出电压，

在短路条件下即$V=0$,此时$I{ph}=I{sc}$, $I{sc}$即短路电流。在开路条件下即$I=0$,此时$V=V{oc}$, $V{oc}$即开路电压
填充因子:
    $FF=\frac {I{max}V{max}}{I{sc}V_{oc}}$

测量串联电阻

在光照条件下通过并联电阻的电流远小于光电流，因此可以忽略不计，
上式可简化为：

$I=I_{ph}-I_0[exp(\frac {q(V+IR_s)}{nKT})-1]$
由于$exp(\frac {q(V+IR_s)}{nKT}>>1$故可以进一步简化为：
$I=I_{ph}-I_0exp(\frac {q(V+IR_s)}{nKT})$

令$\lambda =\frac {q}{nKT}$则：

$V=\frac {1}{\lambda}ln(\frac {I_{ph}-I}{I_0})-IR_s$

当接近开路时即$I \rightarrow 0$, $(I{ph}-I) \rightarrow I{ph}$，电压的表达式为:

$V= \frac {1}{\lambda}ln(\frac {I_{ph}}{I_0})-IR_s$

在开路条件下，电池出输出电压等于开路电压，即$V=V_{oc}$,电池输出电流等于0,则：

$V_{oc}=\frac {1}{\lambda}ln(\frac {I_{ph}}{I_0})$ $V=V_{oc}-IR_s$

此时可以看出太阳能电池在开路状态下，电压和电流具有很好的线性关系，对I-V曲线求导可得:

$R_s=\frac {dV}{dI}$

测量并联电阻

并联电阻通常在暗态下，反向偏置I-V曲线。
计算曲线的线性部分斜率可以得到并联电阻：

$r_{sh}=\frac {\Delta V_{reverse bias}}{\Delta I_{reverse bias}} $
